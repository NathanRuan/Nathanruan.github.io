My bootstrap notes

Bootstrap is the world's most popular front-end open source toolkit, with support for Sass variables and mixins, a responsive grid system, lots of pre-built components and powerful JavaScript plugins to help you quickly design and customize responsive, mobile-first sites.

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Bootstrap has a large number of packaged tools and components to help developers build a usable website easily, intuitively and quickly, and bootstrap also has a large number of templates and themes to provide users with the ability to establish the style of their website more quickly

It is worth noting that before using bootstrap, the developer must first enter the code into the html file in the correct order. Otherwise, bootstrap will not take effect in the page build.

Bootstrap allows users to adjust the layout of a web page in a more intuitive way. On the other hand, such a convenient feature is provided by bootstrap, which has a different arrangement of elements than css layout adjustment. It may take some time for users to get used to it before they become proficient in using it.

Don't forget that bootstrap also provides a library of icons for users outside of the code.The Bootstrap Icon Library is a library of SVG icons designed by @mdo and maintained by the Bootstrap Team, which is constantly being enriched.The Bootstrap Icon Library was born out of the need for icons in Bootstrap's built-in components, such as forms, rotators, etc.
